# Button

按钮组件。

### 用法

在 Taro 文件中引入组件

```js
import { XButton } from 'taro-x-ui'
```

在页面中写入组件

```jsx
<XButton color="red">默认</XButton>
```

### 参数

| 属性  | 说明     | 类型   | 是否必传 | 默认值 |
|-------|----------|--------|----------|--------|
| color | 文字颜色 | string | 否       | #333   |
